## 💻 Installation & Usage

1. **Clone the repository**
   ```sh
   git clone https://github.com/Sakshi-2508/Anime-Website-Bootstarp.git
   ```
2. **Navigate to the project directory**
   ```sh
   cd Anime-Website-Bootstarp
   ```
3. **Open `index.html` in your browser**
